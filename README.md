# Team Agency Portfolio (OJT 2026 Capstone Project)

Welcome to the **Team Agency Portfolio**, a college capstone project built by a team of student developers. This repository serves as a unified dashboard linking six independent projects built entirely with vanilla **HTML5, CSS3, and JavaScript** (no external frameworks or libraries).

## 🚀 Projects Overview

Each project in this portfolio demonstrates key frontend engineering skills, including DOM manipulation, state management, asynchronous programming (Fetch API), and local storage persistence.

1. **Team Portfolio Dashboard** (Built by *Ritesh Yadav* | Branch: `main-ritesh`)
   - The central landing and navigation portal with dynamic theme switching and collapse states.
2. **Interactive Quiz App** (Built by *Dhruv Sharma* | Branch: `main-dhruv`)
   - An interactive quiz application featuring score tracking, progress visualization, and instant feedback.
3. **Expense Tracker** (Built by *Soham Patel* | Branch: `main-soham`)
   - A fully functional budgeting and CRUD transaction tracker powered by `localStorage` persistence.
4. **Live News Feed** (Built by *Vikash* | Branch: `main-vikash`)
   - A real-time news reading client utilizing asynchronous REST API consumption, caching (5-minute TTL), search, and article sharing.
5. **Kanban Task Board** (Built by *Sumeet* | Branch: `main-sumeet`)
   - A drag-and-drop productivity manager showing advanced visual workflows and task state management.
6. **GitHub Explorer**
   - A tool for searching and inspecting GitHub repositories and developer profiles.

## 🛠️ Technology Stack

- **HTML5:** Semantic layouts and accessible UI structure.
- **CSS3:** Responsive styles, variables, transitions, CSS Grid, and Flexbox.
- **Vanilla JavaScript:** Custom interactivity, event handling, local caching, and integration with public Web APIs.

## 💻 Getting Started

To run the portfolio locally, simply clone the repository and open the root `index.html` file in any modern web browser.
