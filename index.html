About This Project

This is a school capstone project built by a team of 6 students using pure HTML, CSS, and JavaScript — no frameworks or libraries. Each member independently built one full project demonstrating key web skills like DOM manipulation, state management, Fetch API, and localStorage. All six projects are connected through a shared dashboard with sidebar navigation and managed through Git branches for collaborative version control.
